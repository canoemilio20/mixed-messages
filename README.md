# mixed-messages
Codecademy Full-Stack Career Path project: Mixed Messages

## Planning
Objective: a JS that, whenever is run, displays a different message to the user.

Requisites:
- Message displayed via the terminal (no GUI).
- The message should be formed randomly from three pieces of data (three parts that combine).
- My chosen topic is: **Epithet Generator**.

Functionality:
1. The user provides a name upon which creat an epithet.
2. The script generates the epithet randomly, drawing from a pool of adjectives, a pool of metaphores/allusions and a pool of phrases.
3. The epithet is displayed on the terminal, together with the name provided.
